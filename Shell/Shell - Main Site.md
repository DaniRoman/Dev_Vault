## Archivo `.zshrc`

>[!tip] Que es?
>

`~/.zshrc` es tu **archivo de configuración de Zsh**, que es el _shell_ (la “terminal”/intérprete de comandos) que estás usando en macOS.

>[!example] Crear alias

```bash
# 1) Añadir alias "vault" a tu ~/.zshrc  
echo 'alias vault="cd /Users/daniel.roman/Library/CloudStorage/OneDrive-AKOElectromecanicaSA/Dev"' >> ~/.zshrc  
  
# 2) Recargar la configuración  
source ~/.zshrc
```
